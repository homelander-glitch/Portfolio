Chess — Browser, Vanilla JS

Game

Full chess game. No library, no engine — I wrote every rule. The hard part wasn't the UI, it was representing board state cleanly enough that the move rules didn't become spaghetti. Turn state, check detection, checkmate — all from scratch.

Agentic Security Auditor

AI / Agentic

An autonomous AI agent that clones a GitHub repo, runs static analysis in a Docker sandbox, uses an LLM to interpret findings and rewrite vulnerable files, then opens a Pull Request with the fixes and a full security report. Full agentic loop — no human in the middle.

  • Bandit static analysis running sandboxed in Docker
  • LangChain + OpenAI for reasoning, fix generation and report writing
  • GitHub API — creates branch, commits changes, opens PR automatically
